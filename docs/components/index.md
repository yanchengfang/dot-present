# 组件总览

Dot Present 以 **Vue 3** 组件为主，可按需全量注册或按子路径按需引入；另有 **React** 包用于虚拟列表等场景。以下为文档站已收录的索引。

## 通用

- [Button 按钮](/components/common/button) — 操作按钮，多类型与图标
- [Card 卡片](/components/common/card) — 头图 + 摘要 / 插槽正文 + 页脚

## 表单

- [Form 表单](/components/form/form) — 配置化字段与校验

## 导航

- [Tabs 标签页](/components/navigation/tabs) — 多面板切换

## 数据展示

- [LazyLoad 图片懒加载](/components/data-display/lazy-load) — 视口内加载与占位

## 进阶组件

- [VirtualList V2（Vue / JSX）](/components/advanced/virtual-list-v2) — `@dot-present/virtual-list-v2`
